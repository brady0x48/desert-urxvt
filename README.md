# desert-urxvt

Port of [desert.vim](https://github.com/fugalh/desert.vim) by [Hans Fugal](https://github.com/fugalh) for rxvt-unicode terminal. I tried to match everything as accurately as possible

## Screenshot

![desert theme](https://imgur.com/0asnh8D)

## Installation

1. Copy contents from file into `~/.Xresources`
2. Make sure to comment out any other colorschemes
3. Source with `xrdb -merge ~/.Xresources`
4. Restart terminal
